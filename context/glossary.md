# Glossary

- Harness: 이 프로젝트에서 작업을 안전하고 일관되게 수행하기 위한 문서, 플레이북, 완료 기준 묶음.
- Agent: 작업을 수행하는 자동화 주체.
- Playbook: 특정 작업 유형에서 따라야 하는 표준 절차 문서.
- Definition of Done: 작업 종료로 간주하기 위한 최소 기준.
- Regression: 기존에 되던 동작이 변경 후 깨지는 현상.
- PWA: 웹 앱이지만 설치, 오프라인 캐시, 앱 같은 실행 경험을 제공할 수 있는 Progressive Web App.
- Session: 사용자가 시작하고 종료한 공부 또는 휴식의 개별 기록.
- Daily Summary: 하루 동안 누적된 공부 시간과 휴식 시간의 집계.
- Visual Timer: 남은 시간이 줄어드는 이미지 또는 마스크로 표현되는 타이머 UI.
- Local Record: Google 로그인 전 브라우저에 저장되는 임시 세션 기록.
- Sync: 로컬 기록과 Firestore 사용자 데이터를 맞추는 과정.
- EC2: AWS의 가상 서버. 이 프로젝트의 1차 배포 대상.
- Docker: 앱과 실행 환경을 컨테이너 이미지로 묶어 배포하기 위한 도구.
